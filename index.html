Curso de paguina web con analitica de datos

Para una empresa o un emprendimiento, el disponer de un sitio web le brinda credibilidad y confianza frente a potenciales clientes. En este curso aprenderás a crear y diseñar tu propio sitio web elegante y moderno desde cero, con una de las herramientas más usadas: WORDPRESS. No necesitas saber programar.
